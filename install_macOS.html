How to install bitmon on macOS

This tutorial will help you with the macOS instalation of the Bitmon game be sure to check all the steps if you are having trouble with the game instalation.

Downloading the Launcher

First lets enter the download page.

If you don't have your account please create one and verify your email.

Now let's go to the bottom of the page there you'll see the Download button for macOS.

Lets wait for the download to be over

Installing the Launcher

Once the download is over open the .dmg file it should promt you with a window.

In that window you'll need to drag the Bitmon game launcher to your applications folder.

After that you can close the window that poped up

Next you have to open finder and go to your apps folder

Search for the Vibing Launcher and right click and then click in open

Then it will prompt you with a message you just need to click on the open button

Installing the game

Launcher will open and then you only need to click on the update button if you don't have latest version

Once it updates just click on start

Game should start and you should be able to play now
